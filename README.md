## Sobre mim

Estudo **Desenvolvimento de Sistemas na ETEC** e **Inteligência Artificial na FMU**, em paralelo. Escolhi backend porque é onde a lógica e as decisões técnicas realmente importam — e é onde me sinto em casa.

Tenho experiência prática na criação de **APIs REST** e **modelagem de banco de dados do zero**, sem depender de serviços externos. Aprendo continuamente por conta própria e busco minha primeira oportunidade como **estagiário** para aplicar esse conhecimento em um ambiente real.

## 🚀 Projeto em destaque

### [EstudeX](https://github.com/Victor-Picciutti)
> Plataforma educacional que conecta alunos e professores — desenvolvida em equipe de 6 pessoas como TCC.

- API REST construída do zero com **Java + Spring Framework**, sem dependências externas
- Modelagem e integração com **SQL Server**, com toda a lógica de negócio no backend
- Containerização com **Docker** e testes completos via **Postman**
- Projeto com site ativo e implantação completa em andamento
